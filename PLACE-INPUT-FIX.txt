Pixel Key 장소 입력 수정

수정 내용
- 카카오 장소 검색 결과를 선택하지 않아도 장소명을 직접 입력해 저장 가능
- "입력한 장소명 그대로 사용" 버튼 추가
- 선택된 장소 확인 카드 추가
- 좌표가 없는 직접 입력 장소도 Supabase에 place_name으로 저장
- 좌표가 있을 때만 상세 지도 표시

검증
- Next.js 16.2.9 production build 통과
- TypeScript 검사 통과

사용법
1. 장소명 입력
2. 카카오 검색을 쓰려면 검색 후 결과 클릭
3. 검색 없이 저장하려면 "입력한 장소명 그대로 사용" 클릭
