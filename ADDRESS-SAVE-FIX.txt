Pixel Key 주소/저장 수정

1. 장소 검색을 누르면 첫 번째 결과가 자동 선택되어 장소명, 도로명주소/지번주소, 좌표가 채워집니다.
2. 검색 목록에서 다른 장소를 누르면 선택이 변경됩니다.
3. 도로명주소를 직접 입력한 뒤 '주소로 정확히 찾기'를 누를 수 있습니다.
4. 지도를 직접 누르면 해당 좌표의 도로명주소 또는 지번주소를 역지오코딩해 선택합니다.
5. 저장 실패 시 Supabase 컬럼/RLS 문제를 구분해 구체적인 안내를 표시합니다.
6. 반드시 RUN-THIS-IN-SUPABASE.sql을 Supabase SQL Editor에서 한 번 실행해야 합니다.
